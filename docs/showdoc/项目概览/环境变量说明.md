# 环境变量说明

项目通过 `.env` 文件管理敏感配置，使用 `python-dotenv` 加载。

## 配置方式

```bash
# 从模板创建
cp .env.example .env
# 编辑 .env 填入你的配置
```

## 变量列表

### Django 基础配置

| 变量名 | 必填 | 默认值 | 说明 |
|--------|------|--------|------|
| `SECRET_KEY` | ✅ | — | Django 密钥，**生产环境必须修改**，见下方生成方法 |
| `DEBUG` | ❌ | `True` | 调试模式，生产环境设为 `False` |
| `ALLOWED_HOSTS` | ❌ | `localhost,127.0.0.1` | 允许的主机名，多个用逗号分隔 |

### Microsoft OAuth（微软登录）

> 需要在 Azure 门户注册应用后获取，详见 [Microsoft OAuth 配置指南](../../microsoft-oauth-setup.md)

| 变量名 | 必填 | 说明 |
|--------|------|------|
| `MS_CLIENT_ID` | ✅ | Azure 应用程序（客户端）ID |
| `MS_CLIENT_SECRET` | ✅ | Azure 应用密钥值 |
| `MS_REDIRECT_URI` | ✅ | OAuth 回调地址，开发环境：`http://localhost:8000/api/v1/auth/microsoft/callback/` |
| `FRONTEND_OAUTH_CALLBACK` | ✅ | 前端接收 OAuth 结果的地址，开发环境：`http://localhost:5173/#/auth/microsoft/callback` |
| `MS_TOKEN_FERNET_KEY` | ✅ | 用于加密存储 Refresh Token 的 Fernet 密钥，见下方生成方法 |

### 文档同步（可选）

| 变量名 | 必填 | 说明 |
|--------|------|------|
| `SHOWDOC_URL` | ❌ | ShowDoc 服务地址（同步文档用） |
| `SHOWDOC_API_KEY` | ❌ | ShowDoc API Key（向管理员索取） |
| `SHOWDOC_API_TOKEN` | ❌ | ShowDoc API Token（向管理员索取） |

## 生成 SECRET_KEY

```bash
uv run python -c "from django.core.management.utils import get_random_secret_key; print(get_random_secret_key())"
```

## 生成 MS_TOKEN_FERNET_KEY

```bash
uv run python -c "from cryptography.fernet import Fernet; print(Fernet.generate_key().decode())"
```

## 注意事项

- ⚠️ `.env` 文件已在 `.gitignore` 中，**不会提交到仓库**
- 每个开发者维护自己本地的 `.env`
- `.env.example` 是模板文件，**会提交到仓库**供参考
- 生产环境建议通过服务器环境变量注入，不使用 `.env` 文件
