# 统一响应格式

所有 API 返回统一的 JSON 结构，便于前端解析和错误处理。

## 响应结构

```json
{
    "code": 200,
    "message": "success",
    "data": { ... }
}
```

| 字段 | 类型 | 说明 |
|------|------|------|
| `code` | int | HTTP 状态码 |
| `message` | string | 操作结果描述 |
| `data` | any | 业务数据，可以是对象、数组或 null |

## 预定义响应方法

在视图中使用 `ApiResponse` 类：

```python
from mctp_api.mctp_api_core.response import ApiResponse

# 成功响应 (200)
return ApiResponse.success(data={"key": "value"}, message="操作成功")

# 创建成功 (201)
return ApiResponse.created(data=serializer.data)

# 未找到 (404)
return ApiResponse.not_found(message="资源不存在")

# 无权限 (401)
return ApiResponse.unauthorized()

# 禁止访问 (403)
return ApiResponse.forbidden()

# 自定义错误
return ApiResponse.error(message="参数错误", code=400)
```

## 响应示例

### 成功

```json
{
    "code": 200,
    "message": "service is running",
    "data": {
        "status": "healthy",
        "version": "1.0.0"
    }
}
```

### 错误

```json
{
    "code": 401,
    "message": "token 已过期",
    "data": null
}
```
