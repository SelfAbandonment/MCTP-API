# Bug 报告模板

在 GitHub Issues 中提交 Bug 时，会自动弹出以下模板：

---

## 需要填写的内容

| 项目 | 说明 |
|------|------|
| **Bug 描述** | 简要描述遇到的问题 |
| **复现步骤** | 1. 做了什么 2. 发生了什么 |
| **期望行为** | 你期望发生什么 |
| **实际行为** | 实际发生了什么 |
| **截图/日志** | 相关截图或错误日志 |
| **环境信息** | OS、Python 版本、Django 版本 |

## 示例

> **Bug 描述：** 用户登录时返回 500 错误
>
> **复现步骤：**
> 1. 访问 /api/login/
> 2. 提交正确的用户名和密码
> 3. 返回 500 Internal Server Error
>
> **期望行为：** 返回 200 和用户 Token
>
> **环境：** Windows 11 / Python 3.12 / Django 6.0.3
